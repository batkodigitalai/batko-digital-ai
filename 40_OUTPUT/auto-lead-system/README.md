# Auto Lead System — GitHub Pages + Google Sheets

Hotová minimální infrastruktura pro více landing pages.

Obsahuje B2B/B2C landing pages, jednotný `lead-client.js`, `config.js` a Google Apps Script backend `apps_script/lead_webhook.gs`.

Veřejné URL po nahrání do `40_OUTPUT/auto-lead-system/`:

- https://batkodigitalai.github.io/batko-digital-ai/40_OUTPUT/auto-lead-system/
- https://batkodigitalai.github.io/batko-digital-ai/40_OUTPUT/auto-lead-system/landing_B2B.html
- https://batkodigitalai.github.io/batko-digital-ai/40_OUTPUT/auto-lead-system/landing_B2C.html

Postup: vytvořit Google Sheet, vložit Apps Script, nasadit jako Web App, URL vložit do `config.js`.
