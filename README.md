# batko.digital.ai – centrální operační systém

Toto repo slouží jako bezpečný řídicí rozcestník pro projekty batko.digital.ai.

## Hlavní pravidlo

Staré veřejné odkazy se nesmí rozbít.

Proto:
- existující veřejná repozitáře se nemažou,
- existující veřejné soubory se nepřesouvají bez redirectu,
- nové systémy se staví paralelně,
- produkce, archiv a experimenty se oddělují.

## Aktuální repo mapa

### Veřejná produkce / veřejné odkazy
- `batkodigitalai/auto1-kalkulacka` – veřejná AUTO kalkulačka, zachovat URL a funkčnost.
- `batkodigitalai/vyhoreni-protokol` – veřejný projekt, před změnami ověřit odkazy.
- `batkodigitalai/strateg-pro-svobodu-od-dluhu-` – veřejný projekt, před změnami ověřit odkazy.
- `batkodigitalai/batko-digital-ai` – hlavní rozcestník a nový bezpečný operační systém.

### Soukromá pracovní vrstva
- `batkodigitalai/taxi-mozek` – interní systém.
- `batkodigitalai/denik-2025` – interní deník / evidence.
- `batkodigitalai/ai-assets` – interní assety.

## Doporučené vrstvy

- `legacy` – staré věci, které musí zůstat funkční.
- `public` – nové veřejné výstupy.
- `active` – aktuální pracovní projekty.
- `content-engine` – šablony a obsah.
- `automation` – skripty a workflow.
- `pipeline` – datové toky.
- `archive` – historie.
- `lab` – experimenty.
