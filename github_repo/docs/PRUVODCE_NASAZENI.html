4


Nasaď jako Web App




Co teď: „Otevřeš" skript veřejně, aby na něj HTML formulář mohl posílat data. Vytvoříš veřejný „webhook".




    

  1. V editoru vpravo nahoře Nasadit (Deploy) → Nové nasazení (New deployment)
    2. 

  2. Vlevo nahoře v dialogu klikni na ozubené kolečko ⚙️ → vyber Web app
    3. 

  3. Vyplň formulář:

    

    Popis: auto lead v3
    

    Spustit jako: Já (Me)
    

    Kdo má přístup: Kdokoliv (Anyone) ⚠️
    

    

    
POZOR: U „Kdo má přístup" musí být Kdokoliv (Anyone), NE „Kdokoliv s účtem Google". Jinak HTML formulář nedokáže poslat data.

    

    4. 

  4. Klikni Nasadit (Deploy)
    5. 

  5. Pokud znovu chce autorizaci, projdi Advanced → Allow (jako v kroku 3)
    6. 

  6. Objeví se dialog s Web app URL — vypadá takto:

    https://script.google.com/macros/s/AKfycb.../exec
    

    7. 

  7. Klikni Kopírovat u té URL
    8. 





✅ Hotovo: Máš webovou adresu, kam HTML formulář bude posílat data.

Časté zádrhele





❓ V menu „Rozšíření" nevidím „Apps Script"


Asi máš jiný účet (např. firemní Google Workspace) kde admin Apps Script zakázal. Zkus se přihlásit ke svému osobnímu Google účtu.







❓ Po spuštění testLead se nic nestane


Buď nemáš uložený kód (krok 2c — Ctrl+S), nebo neprošla autorizace (krok 3.3). Otevři v editoru „Spuštění" (Executions) vlevo — uvidíš log.







❓ Nasadit → říká „Anyone with Google account"


To je špatně. Klikni na šipku a vyber Anyone (bez „with Google account"). Pokud tam ta volba není, máš firemní účet s omezením — vyžaduje admina.







❓ Google říká „Sign in required"


Nejsi přihlášený do Google v Brave. Otevři accounts.google.com, přihlaš se a zkus znovu.







❓ Nemám čas / nezvládám to


Pošli návod (tento HTML soubor) IT známému/synovi. Trvá to 5 minut a celé je to klikání, žádné programování. Pokud ani to ne — pošli mi screenshot kde jsi zasekl, vyřešíme to spolu.
