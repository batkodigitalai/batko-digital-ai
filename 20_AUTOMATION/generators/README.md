# Generators

Sem budou patřit generátory:

- HTML stránek,
- postů,
- scoring reportů,
- markdown exportů,
- feedů,
- landing pages.

## Doporučený princip

Input:

```plaintext
CSV / JSON
```

Output:

```plaintext
HTML / Markdown / Pages / Posty
```

## Kritické pravidlo

Generátory nesmí:
- mazat veřejné soubory,
- přepisovat produkční URL,
- přepisovat legacy projekty.
