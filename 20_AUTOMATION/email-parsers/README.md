# Email parsers

Vrstva pro zpracování emailů.

Budoucí použití:
- AUTO1 nabídky,
- OPENLANE,
- aukční emaily,
- exporty,
- leady.

## Budoucí flow

```plaintext
Email
↓
Parser
↓
JSON / CSV
↓
Scoring
↓
Feed
↓
Landing page / post
```

## Bezpečnost

Parsers:
- nesmí mazat původní data,
- musí mít archiv,
- musí být auditovatelné.
