# Migration plan

Cíl:
Postupně přesouvat důležité know-how mimo přeplněný kontext chatů do stabilního GitHub systému.

Důležité:
Nic veřejného se nesmí rozbít.

---

# Co zůstává v ChatGPT paměti

Pouze:
- styl komunikace,
- hlavní preference,
- dlouhodobá identita,
- základní projekty,
- důležité principy.

# Bezpečnostní pravidla

- nemaž veřejné repo
- nepřepisuj veřejné URL
- produkce a experimenty odděleně
- vždy archivovat před větší změnou

---

# Praktický režim

Když uživatel řekne:

- "tohle je důležité"
→ uložit systemově

- "tohle nechci ztratit"
→ archivovat

- "tohle budeme používat znovu"
→ šablona / reusable systém

- "tohle je veřejné"
→ produkční režim
