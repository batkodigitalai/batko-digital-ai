# 00_SYSTEM

Centrální řídicí dokumentace pro GitHub infrastrukturu batko.digital.ai.

## Účel

Tato složka drží pravidla, mapy, registry a bezpečnostní postupy.

## Zásady

- Staré veřejné odkazy se nerozbíjejí.
- Produkce se nemíchá s experimenty.
- Nové věci vznikají paralelně.
- Před přesunem veřejné URL musí existovat redirect nebo archiv.
- Každý nový projekt musí mít vlastní README.

## Doporučené soubory

- `repo mapa.md`
- `url registr.md`
- `naming convention.md`
- `bezpecnostni pravidla.md`
- `automatizacni plan.md`
