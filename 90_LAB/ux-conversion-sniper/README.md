# UX Conversion Sniper

Streamlit MVP pro audit a přepis UX microcopy. Zkopírujte `.streamlit/secrets.toml`, doplňte OpenAI API klíč a Stripe Payment Link a spusťte:

```powershell
pip install -r requirements.txt
streamlit run app.py
```

Bez API klíče lze použít tlačítko **Načíst demo**; aplikace zobrazí lokální ukázkový výstup.
